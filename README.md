# 📐 Geometry Calculator (Java)

## 📌 Overview
This is a menu-driven Java console application that performs:

- Circle calculations (Area, Circumference, Diameter)
- Triangle calculations (Area, Perimeter)
- Square & Rectangle calculations
- Basic calculator operations

## 🛠️ Technologies Used
- Java
- OOP Concepts
- Scanner Class

## 🚀 Features
- User-friendly menu system
- Multiple classes for modular design
- Error handling (division by zero)

## ▶️ How to Run
1. Compile:
   javac Geometry_Calculater.java
2. Run:
   java Geometry_Calculater
